# ChronoPixel
ChronoPixel is a lightweight and flexible 2D game engine built for JavaScript, designed to empower developers to create immersive and dynamic gaming experiences. With a focus on simplicity and performance, ChronoPixel provides an intuitive API, powerful rendering capabilities, and a rich set of features, enabling you to bring your game ideas to life.

# Features
Canvas-Based Rendering: Leverage the HTML5 canvas for smooth animations and graphics.
Sprite Management: Easily load, manage, and animate sprites for your characters and environments.
Collision Detection: Implement responsive collision detection to enhance gameplay mechanics.
Event Handling: Simplified event system for keyboard, mouse, and touch interactions.
Modular Architecture: Extendable design to integrate additional functionalities and plugins.
Cross-Platform Compatibility: Create games that run seamlessly on desktops and mobile devices.
